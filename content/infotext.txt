Welcome to the OpenAnno Alpha 2008.01, Keybindings:
--------------
- LEFT = Scroll Left
- RIGHT = Scroll Right
- UP = Scroll Up
- DOWN = Scroll Down
- F1 = Display Help
- F5 = Quicksave
- F9 = Quickload
- F10 = Toggle console on / off
- ESC = Show Pause menu
- T = Toggles grid on / off
- X = enable destruct mode
- P = Pause
- "+" = Speed Up
- "-" = Speed Down
- S = Screenshot


Have fun.
